# LLM Bench Support

LLM Bench is an iOS developer tool for repeatable on-device GPU LLM benchmarking.

For app issues, purchase questions, general feedback, or feature requests, contact:

[dansedano.dev@gmail.com](mailto:dansedano.dev@gmail.com)

When reporting a benchmark or model issue, include:

- Device model and iOS version.
- App version and build number.
- Runtime or model format involved, such as MLX, GGUF, or LiteRT-LM.
- Whether the issue happened during setup, model download/import, benchmark run, results review, export, or purchase.
- Any safe diagnostics you are comfortable sharing.

Do not send private prompts, model responses, model files, or confidential benchmark output unless you intentionally want support to review them.
